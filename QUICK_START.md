# ✦ X Content Sidekick — Quick Start

Get running in under 5 minutes.

## Step 1: Get your xAI API key
1. Go to **[console.x.ai](https://console.x.ai)** → API Keys
2. Create a key (starts with `xai-`)

## Step 2: Load the extension
1. Chrome → `chrome://extensions/`
2. Enable **Developer mode** (top-right)
3. Click **Load unpacked** → select the `x-content-sidekick` folder

## Step 3: Set up your profile
1. Click the extension icon → **X Content Sidekick**
2. Paste your API key
3. Fill in your **Voice Profile**:
   - Professional domain (e.g. "AI, B2B SaaS, product")
   - Voice style + engagement goal
4. Click **Save Settings**

## Step 4: Use it

**To reply to a tweet:**
- Find the **✦ Engage** button on any tweet → sidebar opens
- Pick a mode (Insight / Question / Connect / Challenge / Expand)
- Click **Generate Replies** → edit → **Use Reply**

**To write original content:**
- Click the ✦ button (right edge of screen) → **Create** tab
- Enter topic, format, tone → **Generate Drafts**
- **Post on X ↗** copies and opens the composer

**For strategy advice:**
- Open the **Strategy** tab
- Ask anything: "What should I post this week?", "Make this tweet punchier"

## Troubleshooting

| Problem | Fix |
|---|---|
| ✦ Engage button missing | Refresh the X page |
| "Extension context lost" | Refresh the X tab after reloading the extension |
| API error | Check key and credits at console.x.ai |
| Wrong tweet context | Works best clicking from the tweet detail page for threads |
